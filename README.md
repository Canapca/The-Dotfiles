# The-Dotfiles

This is a repo containing my most important dotfiles and other configs mostly related to hyprland and configs I would never remember to use in case I needed to.

### The Config Dump folders
    These contain configs that are either specific to non-vital software or simply aren't "timeless" enough to be put somewhere else.

### The Tricks and Fixes folder
    This one contains specific fixes for a specific issue. For example a wireplumber fix to stop vencord/discord from hijacking the microphone gain.

### The Themes folder
    In this folder you will find configs and CSS files responsible for customising the system, mostly for hyprland installations.
